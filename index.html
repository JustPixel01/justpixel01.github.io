Herzens Mod

Der Herzens Mod ist ein Gameplay-Feature, das mit Leben, Herzen und Belohnungen arbeitet.

  • Neue Spieler starten mit 10 Herzen
  • Bei Tod verliert man ein Herz
  • Herzen können durch Events oder Items zurückgewonnen werden
  • Mit /hearts sieht man die eigenen Herzen
  • Admins können mit /addheart Spielerherzen vergeben

Server Setup

So richtest du den Server ein:

  1. Minecraft Server 1.20.1 installieren
  2. Fabric Loader einrichten
  3. Mods in den Mods-Ordner legen
  4. server.properties konfigurieren
  5. Server starten

Wichtige Gamerules:

/gamerule keepInventory true /gamerule doImmediateRespawn true
