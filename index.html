❤️ Herzens Mod

Herzens Mod

🪙 Items

Seelen Amulett (Soul Amulet)

  • Behält dein Herz, wenn du in einem Kampf stirbst
  • Kann nicht im Survival erlangt werden – nur für Admin Events

⚡ Befehle

  • /hearts set Spieler 1–3 – Anzahl der Herzen eines Spielers festlegen
  • /hearts get Spieler – Zeigt aktuelle Herzen eines Spielers
  • /hearts settings showCombatTimeRemainingInActionbar true|false – Zeigt Kampfzeit im Aktionsbalken
  • /hearts give Spieler – Gib einem anderen Spieler ein Herz
  • /hearts set Spieler – Ändert Status des verknüpften Herzens
  • /hearts get Spieler – Zeigt Partner & Herz-Status
  • /hearts linked pair Spieler1 Spieler2 – Verknüpft zwei Spieler

⚙️ Gamerules

  • /gamerule sendSoulAmuletInformation true|false – Amulette posten Koordinaten im Chat
  • /gamerule banIfNoHearts true|false – Bannt Spieler ohne Herzen
  • /gamerule allowTradingHearts true|false – Erlaubt/verbietet Herz-Handel
  • /gamerule showDeathScreen true|false – Todesscreen an/aus
  • /gamerule useLinkedHearts true|false – Aktiviert verknüpfte Herzen
🛠️ Server Setup

Server Setup

  1. Download: Lade die Server-Dateien herunter.
  2. Installation: Entpacke die Dateien in einen Ordner.
  3. Start: Starte start.bat (Windows) oder start.sh (Linux/Mac).
  4. Konfiguration: Bearbeite server.properties.
  5. Mods: Stelle sicher, dass alle Mods im mods/-Ordner sind.

👉 Java 17 oder neuer wird benötigt!
